# Subsystems

Functional areas of the code, each spanning files within one or more
[projects](/projects/index.md).

* [Quoting](/subsystems/quoting.md) - turn a syntax tree into the C# that rebuilds it.
* [Templating](/subsystems/templating.md) - `[Template]` → factory; the staged-binding pipeline (the gnarliest area).
* [Matching](/subsystems/matching.md) - pattern-match, capture, and replace over syntax.
* [Decorations](/subsystems/decorations.md) - post-quote declaration markers.
